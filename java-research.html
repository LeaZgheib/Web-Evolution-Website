1. The Birth of JavaScript (1995)

JavaScript was created in 1995 by Brendan Eich at Netscape Communications. It was first called Mocha, then LiveScript, and finally JavaScript. Eich designed it in just 10 days to make web pages interactive inside the browser. JavaScript first appeared in Netscape Navigator 2 and was later standardized as ECMAScript by ECMA International in 1997.

2. What's eval in JavaScript? It was blocked. What policy blocks it?

The eval() function in JavaScript executes a string as code. For example, eval("2 + 2") returns 4. However, using eval() is dangerous because it can run malicious code, so it’s often blocked or restricted. Modern browsers and security systems can block eval() under Content Security Policy (CSP). This policy prevents unsafe code execution to protect against attacks like cross-site scripting (XSS).

3. Does JavaScript short-circuit logical expressions when the result is already determined?

Yes. JavaScript uses short-circuit evaluation for logical operators. In expressions with && (AND) or || (OR), it stops evaluating once the result is known. For example: false && something() won’t call something() because the result is already false. This improves performance and avoids unnecessary operations.

4. How can we implement a method in JavaScript that can accept 1000 arguments?

No, you don’t need 1000 separate methods. In JavaScript, functions can accept a flexible number of arguments using the rest parameter (...args) or the built-in arguments object.

function manyArgs(...args) {
  console.log(args.length);
}
manyArgs(1, 2, 3, ..., 1000); // Works fine

JavaScript automatically handles any number of arguments passed to a function.

5. Does JavaScript support pre and post incrementation?

Yes. JavaScript supports both pre-increment (++x) and post-increment (x++) operators.

Pre-increment (++x): Increases the value first, then returns the new value.
Post-increment (x++): Returns the current value first, then increases it.

let x = 5;
console.log(++x); // 6
console.log(x++); // 6 (then becomes 7)
